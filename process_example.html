Creating a Realistic Forest Themed Window Display

Special projects can require some special attention. Using creative ingenuity and cost-effective materials, we created trees and other elements for a forest-themed NYC storefront. Here's how we did it.

Acquired sono tubes, cut and joined them.

Wrapped tubes in paper to create a bark-like look

Adhered more paper to create additional details and tree bases

Laid paste-covered fabric on top of paper to create a more seamless effect.

Set of trees prepped and ready for detailing.

Used twisted fabric to enhance the details on the trees.
